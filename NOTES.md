## 1. 추세 판별 (Trend)

차트 매매의 출발점. **"추세에 역행하지 않는다"** 가 원칙.

### 이동평균선 정배열 (MA Alignment)
- MA5 > MA20 > MA60 > MA120 → **상승 정배열** (매수 우위)
- MA5 < MA20 < MA60 < MA120 → **하락 정배열** (매도 우위)
- 엉켜있으면 → **횡보** (관망 또는 박스권 매매)

### 추세 판정 기준 (시스템에서 사용)
- 현재가 > MA20 이고 MA20 기울기 양수 → `uptrend`
- 현재가 < MA20 이고 MA20 기울기 음수 → `downtrend`
- 그 외 → `sideways`

### 추세선 (Trend Line)
- **상승추세선**: 저점 2개 이상을 이은 직선. 이탈 시 추세 전환 경계
- **하락추세선**: 고점 2개 이상을 이은 직선. 돌파 시 전환 신호

## 2. 지지/저항 (Support/Resistance)

가격이 반복적으로 멈추는 수평선.

- **지지**: 매수세가 강해지는 가격대 → 이탈 시 급락 위험
- **저항**: 매도세가 강해지는 가격대 → 돌파 시 급등 가능
- **저항이 지지로 전환** (돌파 후 되돌림): 강력한 매수 타이밍

### 피보나치와 결합
- 되돌림 61.8% 지지선 = 중요 매수 후보
- 역할 전환이 일어나는 구간에서 매매 승률이 올라감

## 3. 차트 패턴 14종

시스템이 자동 탐지하는 패턴. 각각 **매수/매도 시그널**과 **확인 조건**이 다름.

### 상승 반전 패턴 (바닥에서 나오면 매수)

| 패턴 | 모양 | 확인 조건 | 목표가 계산 |
|---|---|---|---|
| **double_bottom (쌍바닥)** | W자 | 2번째 저점이 1번째 저점과 유사(±3%), 중간 고점 돌파 | 네크라인 + (네크라인 - 저점) |
| **inverse_head_and_shoulders (역헤드앤숄더)** | 3번째 저점 중앙이 가장 낮음 | 네크라인(두 고점 연결) 돌파 | 네크라인 + 머리 깊이 |
| **cup_and_handle (컵앤핸들)** | U자 컵 + 작은 손잡이 | 컵 우측 저항 돌파 + 거래량 증가 | 컵 깊이만큼 상승 |
| **falling_wedge (하락 쐐기)** | 수렴하는 하락 삼각형 | 상단 추세선 돌파 | 쐐기 시작 가격까지 |

### 하락 반전 패턴 (고점에서 나오면 매도)

| 패턴 | 모양 | 확인 조건 |
|---|---|---|
| **double_top (쌍봉)** | M자 | 2번째 고점 후 중간 저점 이탈 |
| **head_and_shoulders (헤드앤숄더)** | 중앙이 가장 높은 3개 봉 | 네크라인 이탈 |
| **rising_wedge (상승 쐐기)** | 수렴하는 상승 삼각형 | 하단 추세선 이탈 |

### 지속 패턴 (추세 유지)

| 패턴 | 의미 |
|---|---|
| **ascending_triangle (상승 삼각형)** | 수평 저항 + 올라가는 저점선. 저항 돌파 시 매수 |
| **descending_triangle (하락 삼각형)** | 수평 지지 + 내려오는 고점선. 지지 이탈 시 매도 |
| **bull_flag (상승 깃발)** | 강한 상승 후 짧은 조정. 조정 저항 돌파 시 매수 |
| **bear_flag (하락 깃발)** | 강한 하락 후 짧은 반등. 반등 지지 이탈 시 매도 |
| **box_breakout (박스권 돌파)** | 좁은 횡보 구간 벗어남. 돌파 방향으로 매매 |

### 캔들 패턴

| 패턴 | 의미 |
|---|---|
| **three_white_soldiers (적삼병)** | 연속 3개 양봉, 각 종가가 직전보다 높음. 강한 매수 |
| **three_black_crows (흑삼병)** | 연속 3개 음봉. 강한 매도 |
| **gap_up / gap_down (갭 상승/하락)** | 전일 고가/저가를 건너뛰고 시작. 강한 모멘텀 |

### 보조 시그널

| 패턴 | 의미 |
|---|---|
| **ma_alignment** | 이평선 정배열 (추세 강도 지표) |
| **volume_spike** | 평균 대비 2배 이상 거래량. 매집/분산 구분 필요 |

## 4. 거래량 분석

**가격보다 거래량이 먼저 움직인다.**

### 확인해야 할 거래량 시그널

1. **돌파 시 거래량 급증**: 저항 돌파에 거래량이 따르면 진짜 돌파 (거래량 없으면 가짜 돌파 의심)
2. **매집 (Accumulation)**: 가격 횡보 + 거래량 증가 → 대형 매수세 유입
3. **분산 (Distribution)**: 가격 횡보 + 거래량 증가 + 고점에서 → 대형 매도세 이탈
4. **OBV (On-Balance Volume)**: 가격 상승일 거래량은 더하고, 하락일은 빼서 누적. 가격과 방향 다르면 **다이버전스** 주의

### 시스템 판정
- 거래량이 20봉 평균의 1.5배 이상 → `volume_spike`
- 해당 봉이 양봉이면 `accumulation`, 음봉이면 `distribution`

## 5. 엘리어트 파동 이론

**가격은 5개 상승파 + 3개 조정파(ABC)로 움직인다.**

### 상승 5파 구조

```
         5
        /
     3 /
    /\/
   2
  /
1
```

- **1파**: 추세 전환, 조심스러운 매수세
- **2파**: 조정 (1파 시작점 하회 금지)
- **3파**: 가장 강력, 거래량 최대 (**5파 중 가장 짧으면 안 됨**)
- **4파**: 조정 (1파 영역과 과도 중첩 금지)
- **5파**: 마지막 상승, 다이버전스 흔함 (주의!)

### 조정 3파 (ABC)
- A파: 첫 하락
- B파: 반등 (속임수 많음)
- C파: 마지막 하락, 종종 A파와 비슷한 크기

### 실전 활용
- **"3파인지 5파인지"** 를 판단하는 게 핵심
- 3파 진행 중이면 공격적 매수
- 5파 진행 중이면 익절 준비
- ⚠️ 파동은 **확정이 아닌 시나리오**로만 봐라. 후행적으로만 확실해진다.

### 시스템 판정
- pivot 기반 고점/저점 탐지 후 규칙 3개(2파/3파/4파) 통과하면 시나리오 제시
- `current_position`: `wave_1_early`, `wave_2_correction`, `wave_3_or_4`, `wave_5_possible_top`

## 6. 피보나치 되돌림/확장

**황금비(0.618)가 시장 심리에 반영된다는 가설**

### 되돌림 (Retracement) - 조정 구간 매수 목표
최근 저점→고점 상승 후 조정이 어디까지 올지:

| 레벨 | 의미 |
|---|---|
| 23.6% | 얕은 조정 (강한 추세) |
| 38.2% | 일반적 조정 |
| **50.0%** | 심리적 중요 |
| **61.8%** | **황금 되돌림** (가장 중요) |
| 78.6% | 깊은 조정 (추세 위험) |

### 확장 (Extension) - 목표가 설정
돌파 후 얼마나 더 갈지:

| 레벨 | 의미 |
|---|---|
| 127.2% | 1차 목표 |
| **161.8%** | **황금 확장** (가장 중요) |
| 200.0% | 2차 목표 |

### 실전 활용
- 상승 추세 매수 타이밍: **38.2~61.8% 되돌림 존에서 분할 매수**
- 목표가: **127.2%, 161.8% 확장 구간에서 분할 익절**
- 손절: 78.6% 이탈 시 시나리오 파괴로 간주

## 7. 매수 타이밍 · 손절 · 목표가

### 매수 타이밍 3가지 유형

1. **돌파 매수**: 저항 돌파 + 거래량 동반 → 돌파 직후 진입
2. **눌림목 매수**: 상승 추세 + 되돌림 38.2~61.8% 도달 → 지지 확인 후 진입
3. **반전 매수**: 쌍바닥/역헤드앤숄더 등 반전 패턴 확정 → 네크라인 돌파 시 진입

### 분할 매수 (시스템 기본 로직)
- **1차 60%**: 주요 지지선 (피보 38.2% 또는 MA20)
- **2차 40%**: 더 깊은 지지선 (피보 61.8%)
- 평균단가를 낮춰 리스크 분산

### 손절 (Stop Loss)
**감정이 아닌 규칙으로 정한다.**

손절 기준 예시:
- 최근 주요 저점 이탈
- ATR × 2 아래
- 매수가 대비 -5~7% (기법별)
- 시스템: `swing_low × 0.97` 또는 `ATR-based`

### 목표가 (Target)
분할 익절:
- **1차 50%**: 피보 127.2% 확장 또는 직전 고점
- **2차 30%**: 피보 161.8% 확장
- **3차 20%**: 추적 손절 (10MA 이탈 시 청산) ← 추세가 길게 가면 수익 극대화

### 손익비 (Risk/Reward Ratio)
```
RR = (목표가 - 매수가) / (매수가 - 손절가)
```
- **최소 2.0 이상** 아니면 매수하지 마라
- 승률 50%여도 RR 2.0이면 기대값 +50%

## 8. 리스크 관리

**수익은 기법이 만들지만, 생존은 리스크 관리가 만든다.**

### 포지션 사이징

종목당 허용 손실 = 총자본 × 리스크 비율 (보통 1~2%)

```
매수 수량 = 허용 손실 / (매수가 - 손절가)
```

예: 자본 1천만원, 리스크 1% → 허용 손실 10만원
매수가 10,000원, 손절가 9,500원 → 10만원 / 500원 = **200주**

### 종합 원칙
1. **한 종목에 자본의 20% 초과 금지**
2. **동시에 5~7개 이상 포지션 금지** (관리 한계)
3. **연속 3회 손절 시 1주일 휴식** (감정 리셋)
4. **주봉과 일봉 방향이 다를 때는 관망** (시스템의 multi_timeframe 검증)
5. **뉴스/이벤트 앞에는 포지션 축소**

## 9. 실전 체크리스트

매수 전 반드시 확인:

### ✅ 진입 조건 체크리스트
- [ ] 주봉 추세가 상승인가? (상위 타임프레임 정렬)
- [ ] 일봉 이평선이 정배열인가?
- [ ] 패턴이 확인되었는가? (네크라인 돌파 등)
- [ ] 돌파 시 거래량이 평균 이상인가?
- [ ] 피보나치 지지 구간에서 매수하는가?
- [ ] 손절가가 명확히 정해져 있는가?
- [ ] 손익비(RR)가 2.0 이상인가?
- [ ] 포지션 크기가 리스크 1~2% 이내인가?

### ❌ 매수 금지 조건
- 주봉 하락 추세
- 이평선 역배열
- 거래량 없는 돌파 (가짜 돌파 가능성)
- 5파 말기 (엘리어트)

- 손절가까지 거리가 너무 멀어 RR이 낮음
